Trust and Security

Protecting user assets is Lumina's top priority. We implement a multi-layered security framework to proactively prevent potential threats and create a secure investment environment.

Regular Code Audits

We regularly review smart contract vulnerabilities through reputable security audit firms and transparently disclose the results.

Bug Bounty Program

Collaborating with the global white-hat hacker community to proactively discover and reward potential security threats.

Multi-Signature (Multi-sig)

Applying multi-signature wallets to project's key assets and contract management to eliminate insider threats and single points of failure.

Risk Management System

Operating a dedicated team and contingency plans to respond to various external risks such as market volatility and regulatory changes.
